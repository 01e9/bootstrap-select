bootstrap-select

A custom select for @twitter Bootstrap using button dropdown; originally by Silvio Moreto.

Example

Make this:

Become this:

  <select class="selectpicker">
    <option>Mustard</option>
    <option>Ketchup</option>
    <option>Relish</option>
  </select>

You can set different Bootstrap classes via the data-btnstyle attribute:

  <select class="selectpicker" data-btnstyle="btn-primary">
...
  </select>

    <select class="selectpicker" data-btnstyle="btn-info">
...
  </select>

    <select class="selectpicker" data-btnstyle="btn-success">
...
  </select>

      <select class="selectpicker" data-btnstyle="btn-warning">
...
  </select>

      <select class="selectpicker" data-btnstyle="btn-danger">
...
  </select>

      <select class="selectpicker" data-btnstyle="btn-inverse">
...
  </select>

Apply span* class to the selects to set the width.

  <select class="selectpicker span2">
...
  </select>

Add the disabled attribute to the select to apply the disabled class. Alternatively, use the data-btnstyle attribute.

  <select class="selectpicker" disabled>
    <option>Mustard</option>
    <option>Ketchup</option>
    <option>Relish</option>
  </select>

Observation

To properly work on some browsers, you need to add a .clearfix class to your css.

Usage

Create yours <select> and add to them the class .selectpicker, like example below:

    <select class="selectpicker">
      <option>Mustard</select>
      <option>Ketchup</select>
      <option>Relish</select>
    </select>

In your javascript, just add the line:

    $('.selectpicker').selectpicker();

You can also pass a button bootstrap class to your select using the option btnStyle, like:

    $('.selectpicker').selectpicker({
      btnStyle: 'btn-info'
    });
