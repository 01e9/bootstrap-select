Bootstrap-select

A custom select for @twitter bootstrap using button dropdown.

Overview

Bootstrap-select creates custom select-like elements (selectpickers) using themed bootstrap components to ensure that your select elements are themed appropriately within bootstrap.

Getting Started

Bootstrap-select requires bootstrap and jquery. Once you have added bootstrap-select to your project, you simply need to add the selectpicker class to any <select> element you wish to theme.

<select class="selectpicker">
    <option>One</option>
    <option>Two</option>
</select>

CDN

The folks at CDNJS host a copy of the library. Just use these links:
